# API Render

Aplicación React para consumir una API y mostrar datos en pantalla.

## Descripción

Este proyecto carga datos desde un endpoint y los despliega en el navegador. Está diseñado para practicar:
- uso de hooks (`useState`, `useEffect`)
- llamadas a una API
- renderizado condicional
- manejo de estados de carga y errores

## Requisitos

- Node.js >= 16
- npm o yarn

## Instalación

Desde la carpeta del proyecto:

```bash
cd c:\Users\Alumno\Desktop\abel\modulo-3\react\api-render
npm install
```

o con yarn:

```bash
yarn install
```

## Ejecución

Iniciar la aplicación en modo desarrollo:

```bash
npm start
```

o:

```bash
yarn start
```

Abrir en el navegador:

```text
http://localhost:3000
```

## Scripts

- `npm start` / `yarn start`: inicia el servidor de desarrollo
- `npm run build` / `yarn build`: genera la versión de producción
- `npm test` / `yarn test`: ejecuta pruebas (si están configuradas)

## Estructura recomendada

- `src/`
  - `App.jsx` o `App.js`
  - `index.jsx` o `index.js`
  - componentes UI
  - servicios o utilidades para consumir APIs
- `public/`
  - `index.html`

## Uso

1. Iniciar la aplicación.
2. Esperar a que se cargue la información desde la API.
3. Visualizar los datos renderizados en pantalla.

## Notas

- Verifica el endpoint en el componente principal o en los servicios.
- Asegura el manejo de carga y errores si la petición falla.
- Para producción, usar `npm run build` y desplegar la carpeta `build`.
